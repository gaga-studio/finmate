# FinMate 프로젝트 개요 v1.0

## 한 줄 정의

FinMate는 합성 마이데이터 기반 또래 페르소나와 내 금융 상태를 비교하고, 가장 큰 격차를 3개월 시뮬레이션과 오늘의 미션으로 바꾸는 청년 금융 온보딩 서비스입니다.

## 핵심 문제

청년은 금융 지식이 전혀 없어서만 행동하지 못하는 것이 아닙니다. 무엇부터 시작해야 하는지, 지금 움직이면 무엇이 달라지는지, 내 또래는 어떤 루틴을 갖고 있는지를 체감하기 어렵기 때문에 미룹니다.

FinMate는 “정보 제공”에서 끝나지 않고 비교, 예측, 행동 전환까지 이어지는 데모를 보여줍니다.

## 핵심 경험

```text
또래 비교 -> 3개월 시뮬레이션 -> 오늘의 미션 -> 개인정보 안전장치
```

| 단계 | 사용자 경험 | 제품 역할 |
| --- | --- | --- |
| 30초 진단 | 상태, 소득대, 목표, 고민 선택 | 비교 기준 설정 |
| 홈 | 오늘 예산, 자산 상태, 또래 teaser 확인 | 현재 상태 요약 |
| 또래 사례 | 합성 또래 페르소나의 루틴 확인 | 행동 기준 제시 |
| 비교 | 내 상태와 또래 사례의 가장 큰 격차 확인 | `mainGap` 도출 |
| 시뮬레이션 | 3개월 뒤 변화 확인 | 행동 효과 예측 |
| 미션 | 오늘 실행할 작은 행동 생성 | 행동 전환 |
| 개인정보 설정 | 공개 미리보기와 철회 | 신뢰 확보 |

## 데모 데이터 원칙

- 발표 데모는 실제 개인정보나 실제 마이데이터를 사용하지 않습니다.
- `MOCK_MYDATA`는 내 금융 상태를 시연하기 위한 합성 데이터입니다.
- `SYNTHETIC_PERSONA`는 또래 사례를 시연하기 위한 합성 페르소나입니다.
- 향후 실제 서비스에서는 별도 동의, 가명 처리, 마스킹, 철회, audit log가 필요합니다.

## 하나금융 역할

하나금융은 이 데모에서 금융상품을 바로 판매하는 주체가 아니라, 신뢰 가능한 공식 정보, 금융 교육 콘텐츠, 리워드, 장기 청년 온보딩 기반을 제공하는 파트너로 표현합니다.

## Public v1.0 기준

- 공개 P0 API: 12개
- OpenAPI version: `1.0.0`
- 핵심 미션: `이번 달 비상금 자동이체 10만 원 설정하기`
- 핵심 격차: `EMERGENCY_FUND`
- 핵심 시뮬레이션: `0.4개월 -> 0.7개월`
