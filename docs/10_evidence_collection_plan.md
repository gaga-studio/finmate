# FinMate 문제 근거 수집계획

## 원칙

이 문서는 발표 근거 슬라이드를 실제 자료로 채우기 위한 계획입니다. 현재 레포에는 임의 인터뷰 수, 설문 비율, 시장 수치를 넣지 않습니다.

## 수집할 근거

| 근거 종류 | 최소 기준 | 기록 형식 | 상태 |
| --- | --- | --- | --- |
| 청년 인터뷰 | 3명 이상 | 직접 발화, 공통 키워드, 행동을 미룬 이유 | [추가 필요] |
| 간단 설문 | 20명 내외 | 상위 선택지, 응답 비율, 짧은 인사이트 | [추가 필요] |
| 기존 앱 비교 | 금융 앱 3개 이상 | 리포트, 비교, 미션, 동의 철회 UI 비교표 | [추가 필요] |
| 공식 자료 | 2개 이상 | 출처, 핵심 문장, FinMate와의 연결점 | [추가 필요] |

## 인터뷰 질문

1. 최근 3개월 안에 저축이나 비상금 만들기를 미룬 경험이 있나요?
2. 미룬 이유는 지식 부족, 돈 부족, 계기 부족, 귀찮음 중 무엇에 가까웠나요?
3. 내 또래의 실제 루틴을 보면 행동을 시작하는 데 도움이 될까요?
4. 오늘 바로 할 수 있는 금융 미션이 있다면 어떤 수준이 부담 없나요?
5. 익명 또래 사례를 본다면 어떤 정보는 반드시 숨겨야 한다고 생각하나요?

## 설문 항목 초안

| 질문 | 응답 형태 |
| --- | --- |
| 금융 행동을 미루는 가장 큰 이유는 무엇인가요? | 단일 선택 |
| 또래 금융 루틴을 보면 동기부여가 될까요? | 5점 척도 |
| 월 10만 원 자동이체 미션은 실행 가능해 보이나요? | 5점 척도 |
| 개인정보 공개 미리보기/철회가 있으면 안심되나요? | 5점 척도 |
| 금융 앱에서 분석 이후 행동으로 이어지지 않았던 경험이 있나요? | 예/아니오 |

## 기존 앱 비교 기준

| 기준 | 확인 내용 |
| --- | --- |
| 분석 | 소비/자산 리포트가 있는가 |
| 비교 | 또래 또는 코호트 비교가 있는가 |
| 행동 전환 | 오늘 할 미션/루틴으로 이어지는가 |
| 안전 | 공개 미리보기, 마스킹, 철회가 명확한가 |
| 교육 | 공식 정보나 금융 개념 학습으로 연결되는가 |

## 발표 반영 방식

- 실제 수집 전에는 `[추가 필요]`로 둔다.
- 수치를 넣을 때는 출처와 수집일을 함께 적는다.
- 설문 수가 작으면 “탐색적 설문”이라고 표현한다.
- 법률/규제 문서는 서비스 안전 설계 근거로만 사용하고 단정 표현은 피한다.
