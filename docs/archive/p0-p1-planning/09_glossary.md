# FinMate 용어사전

| 용어 | 의미 | 사용 기준 |
| --- | --- | --- |
| FinMate | 청년 금융 온보딩 서비스 | 제품명 |
| P0 | 7분 발표 데모에서 반드시 완주해야 하는 핵심 범위 | API 12개 고정 |
| MOCK_MYDATA | 발표 데모용 합성 마이데이터 | 실제 연동 아님 |
| SYNTHETIC_PERSONA | 발표 데모용 합성 또래 페르소나 | 실제 개인 아님 |
| 또래 사례 | 가명/마스킹된 금융 루틴 카드 | 평균값만이 아니라 행동 루틴을 보여줌 |
| mainGap | 내 상태와 또래 사례의 가장 큰 격차 | P0에서는 `EMERGENCY_FUND` |
| 비상금 준비율 | 현금성 자산 / 월평균 필수지출 | 데모 before `0.4`, after `0.7` |
| 오늘의 미션 | 시뮬레이션 결과를 오늘 실행할 행동으로 바꾼 카드 | P0는 자동이체 10만 원 미션 |
| privacyBadges | 가명 처리, 거래처 비공개, 구간값 표시 등 안전 표시 | 포트폴리오 상세에서 노출 |
| privacySharePreview | 친구 공유 시 금액이 포함되지 않는지 확인하는 미리보기 | P0는 `containsAmount=false` |
| Social Impact | 청년이 작은 금융 경험을 통해 이해도를 높이는 확장 가치 | Birthday Wish Money는 P2 |
| Birthday Wish Money | 생일 선물을 목표 금액 설정, 소액 적립, 달성 경험으로 바꾸는 확장 기능 | P0 구현 제외 |
| 하나금융 역할 | 공식 정보, 교육, 리워드, 신뢰 기반 제공 | 판매자 중심 표현 금지 |
